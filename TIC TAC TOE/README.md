# Tic Tac Toe Game

A classic two-player Tic Tac Toe game with a modern, responsive design. Play with friends on the same device.

## Features

- Two Player Mode - Play with a friend on the same device
- Custom Player Names - Personalize with your own names
- Smart Game Logic - Automatic win detection and tie handling
- Turn Indicator - Clear visual of whose turn it is
- Winner Highlight - Winning cells are highlighted
- Responsive Design - Works on desktop, tablet, and mobile
- Modern UI - Clean design with smooth animations

## How to Play

1. Enter names for Player 1 (O) and Player 2 (X)
2. Click "Start Game" to begin
3. Players take turns clicking on the grid
4. First to get three in a row wins
5. If all 9 squares are filled, it's a tie
